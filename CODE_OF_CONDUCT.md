# Code of Conduct

## Our Pledge

We want formkitcn to be a welcoming, inclusive project where anyone — regardless of experience level, background, identity, or any other factor — feels comfortable contributing and participating.

## The Short Version

Be kind. Be respectful. Assume good intentions.

That's really it. But if you want the details, read on.

## The Longer Version

We're committed to providing a harassment-free experience for everyone. We do not tolerate:

- Sexist, racist, homophobic, transphobic, ableist, or otherwise discriminatory jokes or language
- Personal attacks or insults
- Trolling or deliberate intimidation
- Publishing others' private information without permission
- Sustained disruption of discussions
- Any other conduct that would be considered inappropriate in a professional setting

## Where This Applies

This code of conduct applies to all project spaces — GitHub issues, pull requests, discussions, and any other communication channels associated with formkitcn.

## Enforcement

If you experience or witness behavior that violates this code of conduct, please report it by opening an issue or contacting the maintainers directly. All reports will be handled with discretion.

Project maintainers have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this code of conduct.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

**tl;dr — Don't be a jerk. We're all here to build cool stuff and help each other out.**
